# 🚀 AI Job Application Tracker

A **Streamlit-based web application** that helps users manage and track their job applications in one place.
The application allows users to **add, update, delete, and analyze job applications** through an interactive dashboard.

Built with **Python, SQLite, and Streamlit**, the project demonstrates full-stack development skills including database management, CRUD operations, analytics visualization, and deployment.

---

## 🌐 Live Demo

🔗 https://ai-job-application-tracker.streamlit.app

---

## 📌 Features

* ➕ **Add Job Applications**

  * Enter company name, role, application status, and application date.

* 📋 **View Applications**

  * Displays all stored applications in a clean interactive table.

* 🔄 **Update Status**

  * Update job progress such as:
  * Applied
  * Interview
  * Rejected
  * Offer

* ❌ **Delete Applications**

  * Remove incorrect or outdated job records.

* 📊 **Application Analytics**

  * Visualize job search progress with charts.

## ⚡ Installation

Clone the repository:

```
git clone https://github.com/gaurimk/ai-job-application-tracker.git
```

Move into the project folder:

```
cd ai-job-application-tracker
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the Streamlit application:

```
streamlit run dashboard.py
```

---

## 📦 Deployment

This project is deployed using **Streamlit Community Cloud**.

Steps to deploy:

1. Push project to GitHub
2. Go to https://share.streamlit.io
3. Connect your GitHub repository
4. Select `dashboard.py` as the main file
5. Deploy
